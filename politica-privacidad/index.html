Política de Privacidad

Última actualización: 14 de febrero de 2026

1. Responsable del tratamiento

El responsable del tratamiento de los datos recogidos en este sitio web es:

  • Nombre: Andrés Sánchez Botta
  • Sitio web: andresbotta.dev
  • Email: contacto@andresbotta.dev
  • Ubicación: Lima, Perú

2. Datos que recopilamos

Este sitio web es de carácter informativo y no recopila datos personales de forma directa a través de formularios de registro o inicio de sesión. Los datos que pueden recopilarse son:

  • Datos de navegación: Dirección IP, tipo de navegador, sistema operativo, páginas visitadas, tiempo de permanencia y otros datos técnicos proporcionados automáticamente por tu navegador.
  • Datos de contacto voluntario: Cuando nos contactas voluntariamente a través de WhatsApp o email, recopilamos los datos que nos proporcionas (nombre, email, mensaje).

3. Finalidad del tratamiento

Los datos recopilados se utilizan exclusivamente para:

  • Mejorar la experiencia de navegación del sitio web.
  • Responder a consultas y solicitudes de cotización.
  • Analizar el uso del sitio web de forma agregada y anónima.

4. Cookies

Este sitio web puede utilizar cookies técnicas necesarias para su funcionamiento. No utilizamos cookies de terceros con fines publicitarios. Las cookies técnicas se eliminan al cerrar el navegador.

5. Compartir datos con terceros

No vendemos, comercializamos ni transferimos tus datos personales a terceros. Los únicos servicios de terceros que pueden procesar datos de forma anónima son:

  • Google Fonts: Para la carga de tipografías (puede registrar la dirección IP).
  • Cloudflare CDN: Para la entrega de recursos estáticos.

6. Derechos del usuario

Tienes derecho a:

  • Acceder a tus datos personales.
  • Rectificar datos inexactos.
  • Solicitar la eliminación de tus datos.
  • Oponerte al tratamiento de tus datos.

Para ejercer cualquiera de estos derechos, puedes contactarnos en contacto@andresbotta.dev.

7. Seguridad

Implementamos medidas de seguridad técnicas para proteger los datos personales contra acceso no autorizado, alteración, divulgación o destrucción. El sitio web utiliza protocolo HTTPS para cifrar las comunicaciones.

8. Cambios en esta política

Nos reservamos el derecho de actualizar esta política de privacidad en cualquier momento. Los cambios serán publicados en esta página con la fecha de la última actualización.
